# Master Campaign Calendar — Aethelmark

**Purpose:** Central reference point for all Aethelmark campaigns and scenarios. Shows when each scenario takes place and what major world events have or haven't occurred as of that date.

This allows different sessions and scenarios to branch off at different timeline points, with consistent event tracking.

---

## How to Use This Calendar

1. **Starting a new scenario:** Note the start date in the scenario file and on this calendar
2. **During a checkpoint:** Reference this calendar to answer "has event X happened yet?"
3. **Between scenarios:** Advance the calendar and update what events have occurred
4. **Branching timelines:** If a scenario diverges from expected outcomes, update the affected threads on this calendar

## Using This Calendar in Sessions

### When Starting a Checkpoint
Reference the Master Calendar to answer:
- What date is it today?
- What major events have already happened?
- What events are still pending?
- What time-sensitive events are approaching?

**Example:** Checkpoint on 12 April 1650 during Viktor scenario:
- Oswin embezzlement: **HAPPENED** (2 days ago)
- Voss & Kraemer: **IN TRANSIT** (summoned 2 days ago)
- Aldren meeting: **UPCOMING** (likely within 1-2 days)
- Judge Kinsky birth: **APPROACHING** (due in 2-3 weeks)

### When Multiple Sessions Occur at Different Times
Keep this calendar updated so each session knows what world-state to assume.

**Example:** If Viktor scenario runs through 20 April, then Sergovy scenario begins 22 April, Sergovy's checkpoint will reference:
- Oswin situation: **ONGOING** (12 days into confinement)
- Corvel investigation: **ONGOING** (constable acting on it for 12 days)
- Judge Kinsky: **DUE IMMINENTLY** (days away from birth)

## Timeline Continuity Notes

**World Assumptions:**
- Aethelmark follows a single shared timeline (1650 onward)
- Seasons progress naturally: January → December
- Multiple scenarios can occur simultaneously (Viktor + Sergovy in April) or sequentially
- Major events propagate: News about Viktor's pregnancy, the Corvel investigation, Judge Kinsky's birth, etc. will become known in the region and affect other scenarios

**Event Propagation:**
- Scandal at Steinfeld (embezzlement) → gossip reaches Isalia's Manor
- Judge Kinsky's birth → court knowledge, will be discussed at noble gatherings
- Sergovy arriving at Manor → servants talk, staff awareness spreads

This is a living world. Events in one scenario shape the context for others.
